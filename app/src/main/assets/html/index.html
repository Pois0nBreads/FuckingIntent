使用方法

目前跳转器支持三种跳转方法

方法一（链接直接跳转）

在浏览器打开
"m.tieba.com/f?tid=帖子id"
"tieba.baidu.com/p/帖子id"
这种链接时会自动跳转到贴吧
（只有点击链接才会响应）

方法二（使用浏览器分享功能跳转）

进入贴子网页后点击浏览器的分享按钮，使用
"百度贴吧"打开即可

方法三（使用通知快捷按钮跳转）

此方法仅适用于Android N(7.0) 以上版本使用

在通知栏处添加 "用贴吧打开链接" 的快捷按钮。
之后只要复制帖子的链接后点击这个按钮即可跳转
